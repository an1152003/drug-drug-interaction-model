# Drug drug interaction model
Tương tác thuốc–thuốc có thể gây hậu quả nghiêm trọng, nên việc dự đoán là rất quan trọng. Các phương pháp truyền thống tốn thời gian, vì vậy học máy được ứng dụng để phân tích dữ liệu lớn và cải thiện dự đoán. Kết hợp nhiều nguồn dữ liệu và kỹ thuật ensemble giúp tăng độ chính xác và hỗ trợ quyết định lâm sàng.
